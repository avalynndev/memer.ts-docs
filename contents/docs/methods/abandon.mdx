## .abandon(text);
Generates a Meme on abandon.

| PARAMETER   | TYPE   | DESCRIPTION                                    |
| ----------- | ------ | ---------------------------------------------- |
| text        | string | Text to Generate meme                          |

Scope: `ASYNC`

Returns: A [`Promise`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise) that resolves to a [`Buffer`](https://nodejs.org/dist/latest/docs/api/buffer.html#buffer_class_buffer).
