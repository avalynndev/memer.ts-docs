## .byemom(avatar, username, text);
Generates a Meme on Bye Mom.

| PARAMETER   | TYPE   | DESCRIPTION                                    |
| ----------- | ------ | ---------------------------------------------- |
| avatar      | string | Avatar URL to Generate welcome image           |
| username    | string | Username of the user to Generate welcome image |
| background  | string | Text to Generate meme                          |

Scope: `ASYNC`

Returns: A [`Promise`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise) that resolves to a [`Buffer`](https://nodejs.org/dist/latest/docs/api/buffer.html#buffer_class_buffer).
